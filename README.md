# 대외활동 큐레이션

대학생 대외활동·공모전·장학·봉사 정보는 링커리어·위비티·1365·장학재단·지자체 등
**여러 곳에 흩어져** 있고, 각 사이트는 "모든 공고"를 보여줄 뿐 **내가 실제로 지원 가능한 것**은
직접 골라내야 한다. 이 프로젝트는 거기서 출발한다.

## 무엇을 만드나 (그리고 무엇을 약속하지 않나)

- ✅ **핵심 가치 = 개인 자격 매칭.** 내 조건(학년·전공·지역·소득분위·자격 등)을 입력하면
  **내가 지원 가능한 것만** 골라준다. (기존 사이트가 안 해주는 부분)
- ✅ **엄선한 소스는 빠짐없이.** 특정한 소스들에 대해선 누락 없이 수집한다.
- ❌ **"인터넷 전체 총망라"는 약속하지 않는다.** 세상 모든 공고를 다 덮는 건 누구도 못 하며,
  못 지킬 약속이다. 대신 **커버하는 소스를 투명하게** 밝힌다.
